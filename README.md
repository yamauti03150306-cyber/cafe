# 焚人旅人 Webサイト（進級制作）

## 作品概要

愛知県あま市にある自家焙煎カフェ「焚人旅人」のWebサイトを制作しました。  
営業日やイベント情報を分かりやすく伝えることを重視しながら、店内の落ち着いた雰囲気が伝わる色使いやデザインを意識しています。  
地元の方が日常の中でふと立ち寄りたくなるような、情報の見やすさとお店の魅力が両立したWebサイトを目指しました。

## アピールポイント

- 初めて訪れる方でもひと目で内容が分かる構成
- 店舗の雰囲気・メニュー・アクセス情報を直感的に把握できる導線設計
- スクロールに合わせて自然と情報が目に入るレイアウト
- 手書き風フォントを効果的に使用し、温かく柔らかな印象を演出

## 制作を通して（感想）

今回の長期間にわたる進級制作を通して、自らクライアントに依頼し、Webサイトを一から制作することの大変さを実感しました。  
Figmaで考えたデザインをコーディングで形にしていく中で、試行錯誤を重ねる場面も多くありましたが、その分、実践的な経験を積むことができました。  
この経験を、今後の学びや制作活動にも活かしていきたいと考えています。
