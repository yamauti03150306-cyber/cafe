# 焚人旅人 公式サイト

このリポジトリは、オーガニックコーヒーと自家焙煎スイーツのカフェ「焚人旅人」の公式Webサイト（静的HTML・CSS・画像等）を管理するためのものです。

##サイト概要

**焚人旅人** は、木のぬくもりある雰囲気の空間で  
厳選したオーガニック豆のコーヒーや手作りスイーツを楽しめるカフェです。  
訪れるすべての人が穏やかなひとときを過ごせるよう、やさしい時間を大切にしています。

公開URL：https://yamauchi0701.upper.jp/sinkyu/

---

## ページ構成

| ファイル | 内容 |
|------|------|
| index.html | トップページ |
| menu.html | メニュー紹介 |
| news.html | お知らせ |
| access.html | アクセス情報 |
| assets/ | 画像 / CSS / JavaScript など |

---

## 主な特徴

- HTML / CSS による静的サイト
- 写真を大きく使用し、カフェの雰囲気を重視
- 地図と営業情報を分かりやすく配置
- シンプルで迷わない導線設計

---

## 開発環境

- HTML5  
- CSS3  
- JavaScript（必要に応じて）

特別なビルド環境は不要です。

### ローカルでの確認方法

1. リポジトリをクローン
   ```
   git clone <repository-url>
   ```
2. index.html をブラウザで開く
